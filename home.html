Overview

The book store API created to provide information for Developers on how intract with the book store service.
provided requests: GET / POST / DELETE . in case of POST user can send JSON file with the supplied structure .
The result will return in JSON File.
if the URL is not found you will get 404 response .
in case of an error in a JSON you will get JSON file with the error (validation ec.) and a 500 response.
in case of success you will get json with success message and 200 response

Get Started

get all books function

you can find all books with this request:

https://computer-store-service.herokuapp.com/books GET request
_id: you can use id for GET resquest to get specific book
and because developers are always bored you can just press
the link to arive to the book page with this specific id

the result is a JSON in this sturture: 

 
                    

{
    "books": [{
        "_id": "5b00b8fab76dd657d80e261d",
        "name": "c/c++",
        "price": 20,
        "author": {
            "name": "mahmod",
            "awards": "winnner"
            },
            "request": {
                "types": "GET",
                "url": "https://computer-store-service.herokuapp.com/books/5b00b8fab76dd657d80e261d"
                }
        }]
}

Create new book function

you can create a new book:

https://computer-store-service.herokuapp.com/books POST request
you must supply a name and price and author name and wards
you must fill all feilds (all feilds are required)

the JSON you supply must be in this sturture: 

 
                        

{
    "name": "introduction to algorthims",
    "price": 15,
    "author": {
        "name": "mahmod",
        "awards": "best education book"
    }
}

the JSON you get in case of success : 

 
                                

{
    "message": "book created successfully",
    "createdBook": {
        "author": {
            "name": "mahmod",
            "awards": "best education book"
            },
        "_id": "5b018f6698e64740682a303c",
        "name": "introduction to algorthims",
        "price": 15,
        "__v": 0
    }
}
