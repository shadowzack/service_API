Overview

The book store API created to provide information for Developers on how intract with the book store service.
provided requests: GET / POST / DELETE / PUT . in case of POST user can send JSON file with the supplied structure .
The result will return in JSON File.
if the URL is not found you will get 404 response .
in case of an error in a JSON you will get JSON file with the error (validation ec.) and a 500 response.
